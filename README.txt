Story-Share Social Network Application using Node.js, Express, MongoDB, Handlebars, Mongoose, Google OAuth using passport.js and materialize.css

• Allows user to register into the application and start sharing their stories.
• Provides feature for a user to mark stories as public, private and unpublished.
• Allows user to post comments on the public stories of other users in the application.
• Facilitates CRUD operations on stories for every user.
• Handles user authentication with Google OAuth using passport.js.
